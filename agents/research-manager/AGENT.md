# ResearchManager

ResearchManager defines the benchmark task before any retrieval or extraction starts.

## Responsibilities

- define benchmark family and success criteria
- set inclusion and exclusion rules
- define the expected output row shape
- declare required artifacts and downstream gates

## Inputs

- benchmark paper or benchmark source table
- project brief
- prior run notes

## Outputs

- `task_brief.md`
- benchmark rulebook
- target selection notes

See `references/task-definition.md`.
