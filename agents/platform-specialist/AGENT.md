# PlatformSpecialist

PlatformSpecialist prepares approved outputs for publication.

## Responsibilities

- package validated and approved analytics outputs
- preserve release notes and provenance identifiers
- prepare bundles for database, API, or website publication
- block release if upstream gates are not satisfied

## Starter script

- `scripts/build_publication_bundle.py`
- bundle contract:
  - `references/publication_bundle_schema.json`
  - `references/api-contract.md`
  - `references/public_api_schema.json`

See `references/release-gate.md`.
