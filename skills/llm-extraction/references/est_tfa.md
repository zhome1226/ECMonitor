# EST / TFA Profile

Use this profile when a benchmark source table is the row-level truth.

## Priority

1. benchmark source table is the only row source
2. article references resolve the full citation
3. mapping documents resolve DOI and URL
4. local PDFs or Zotero only help confirm matches

## Core rules

- do not replace source-table values with narrative values
- do not use the benchmark article DOI as the source DOI
- preserve unresolved references explicitly
