# Nature Profile

Use this profile for record-level monitoring reconstruction.

## Extraction level

- one pollutant
- one matrix
- one location
- one time slice
- one value or one range

## Priorities

1. supplementary tables
2. main-text tables
3. main-text results
4. figures only if values are explicit

## Never extract

- toxicity thresholds
- removal experiment doses
- modeled outputs
- QA/QC thresholds as concentration records
